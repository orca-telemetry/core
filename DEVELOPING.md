## Developer Guide

### Building the Protobufs

_Nodejs_:

- `npm install --global protoc-gen-ts`
- `npm install --global grpc_tools_node_protoc_plugin`
- `npm install --global ts-protoc-gen`
- `npm install --global google-protobuf`

_Python_:

- `python -m install grpc_tools`
